<h1 align="center">
VS Code Red Alerts
</h1>

This extension will display red alerts and play a sound.

## Getting Started

- Add the areas you want to get notified about in the settings("redAlerts.areas").
  >
  > <ins>settings.json</ins>
  > ```json
  > {
  >     "redAlerts.areas": ["אשקלון", "אשדוד"]
  > }
  > ```
